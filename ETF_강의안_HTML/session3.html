데이터 수집 실행 + 대시보드 제작

⏱️ 35분
🎯 이 회차를 마치면: 유튜브에서 ETF 데이터를 진짜로 모아오고, 그 데이터를 예쁜 대시보드로 만들어 브라우저에서 볼 수 있어요! 핵심 프로젝트 완성!

Step 1. 필요한 재료(라이브러리) 설치 (3분)

Git Bash에서 프로젝트 폴더로 이동하세요:

cd ~/etf-dashboard

프로그램에 필요한 재료를 설치합니다:

pip install -r requirements.txt
📌 "requirements.txt에 적힌 재료들을 전부 설치해줘"라는 뜻. 설치에 30초~1분 걸려요.

Step 2. 데이터 수집 프로그램 실행! (5분)

드디어! 프로그램을 실행해봅시다:

python collect_data.py

화면에 이런 진행 상황이 나타날 거예요:

"ETF 투자" 검색 중...
  10개 영상 수집 완료
"ETF 추천" 검색 중...
  10개 영상 수집 완료
"ETF 분석" 검색 중...
  10개 영상 수집 완료

총 30개 영상 데이터 수집 완료!
etf_youtube_data.json에 저장했습니다.

🎉 축하해요! 유튜브에서 ETF 데이터를 자동으로 모아왔습니다!

어떤 데이터가 모였는지 살짝 확인해볼까요?

python -c "import json; data=json.load(open('etf_youtube_data.json',encoding='utf-8')); print(f'총 {len(data)}개 영상'); print(data[0]['title'])"

첫 번째 영상의 제목이 화면에 나와요!

Step 3. Claude에게 대시보드 만들어달라고 부탁 (12분)

📌 대시보드가 뭔가요?
자동차 계기판(dashboard)처럼, 중요한 숫자와 그래프를 한 화면에 보여주는 웹 페이지!

Claude Code를 실행하세요:

claude

아래 내용을 통째로 복사해서 붙여넣으세요:

etf_youtube_data.json 파일에 유튜브 ETF 영상 데이터가 있어.
이 데이터를 보여주는 예쁜 웹 대시보드를 HTML로 만들어줘.

요구사항:
1. 하나의 index.html 파일로 만들어줘 (서버 없이 바로 열 수 있게)
2. 한국어로 작성
3. 반드시 포함할 내용:
   - 상단에 "ETF 유튜브 트렌드 대시보드" 제목
   - 전체 영상 수, 총 조회수, 평균 좋아요 수 요약 카드
   - 조회수 TOP 10 영상 막대 그래프 (Chart.js 사용)
   - 채널별 영상 수 파이 차트
   - 전체 영상 목록 테이블 (제목 클릭하면 유튜브로 이동)
4. 디자인: 깔끔한 다크 테마, 카드형 레이아웃
5. JSON 데이터를 HTML 파일 안에 직접 포함시켜줘
6. 모바일에서도 보기 좋게 반응형으로 만들어줘

Enter! Claude가 대시보드를 만들기 시작합니다.

💡 파일 만들어도 되는지 물으면 "y" 입력. 코드 작성에 1~3분 걸릴 수 있어요!

Step 4. 브라우저에서 대시보드 열기! (5분)

Claude가 작업을 마치면:

/exit
ls

index.html 파일이 보이면 성공!

🎉 이 순간을 위해 여기까지 왔습니다!

start index.html

인터넷 브라우저가 열리면서 여러분이 만든 대시보드가 나타납니다!

화면에 이런 것들이 보여요:

• 맨 위: "ETF 유튜브 트렌드 대시보드" 제목
• 요약 카드: 전체 영상 수, 총 조회수, 평균 좋아요 수
• 막대 그래프: 조회수 TOP 10 영상
• 파이 차트: 채널별 영상 수
• 영상 목록 테이블: 제목 클릭하면 유튜브로 이동!

Step 5. 대시보드 꾸미기 - 도전 과제! (5분)

마음에 안 드는 부분이 있다면? Claude에게 수정을 부탁하세요!

claude
> index.html의 배경색을 좀 더 밝게 바꿔줘
> 그래프 색상을 파란색 계열로 바꿔줘
> 조회수 순 정렬 기능을 추가해줘
💡 수정 후 브라우저에서 Ctrl+R(새로고침)을 누르면 변경 확인!

✅ 체크포인트

  • python collect_data.py 실행 시 "수집 완료" 메시지가 나왔나요?
  • etf_youtube_data.json 파일이 만들어졌나요?
  • index.html 파일이 만들어졌나요?
  • 브라우저에서 대시보드가 열렸나요?
  • 그래프와 표가 화면에 보이나요?

🚨 막혔을 때

증상원인해결법
ModuleNotFoundError재료 미설치pip install -r requirements.txt 다시 실행
API key invalid / 403 에러API 키 문제.env 파일 키 재확인 + Google Cloud에서 API "사용" 상태 확인
인코딩 에러 (UnicodeDecodeError)한글 처리 문제Claude에게 "인코딩 에러 나요. utf-8로 수정해줘" 부탁
브라우저에 아무것도 안 보여요HTML 파일 문제Claude에게 "index.html이 안 열려요. 확인해줘" 부탁
그래프가 안 보이고 빈 화면데이터 미포함Claude에게 "그래프 안 보여요. 데이터 다시 확인하고 HTML에 넣어줘" 부탁
start 명령이 안 먹어요명령어 문제etf-dashboard 폴더에서 index.html 파일을 직접 더블클릭
