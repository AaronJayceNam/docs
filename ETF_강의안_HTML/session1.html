터미널부터 AI 비서까지 한 번에!

⏱️ 40분
🎯 이 회차를 마치면: 터미널, Git, Node.js를 설치하고, AI 비서 Claude Code와 첫 대화까지! "와, 이런 거구나!" 체험을 바로 해볼 수 있어요.

Step 1. Microsoft Store에서 Windows Terminal 설치 (5분)

키보드에서 윈도우 키(키보드 왼쪽 아래 창문 모양)를 한 번 누르세요.

검색창이 뜨면 아래 글자를 그대로 입력하세요:

Microsoft Store

파란 쇼핑백 모양 아이콘이 보이면 클릭하세요.

Microsoft Store가 열리면, 위쪽 검색창에 입력하세요:

Windows Terminal

"Windows Terminal" 앱을 찾아 "설치" 또는 "다운로드" 버튼을 누르세요.

💡 TIP: 이미 "열기" 버튼이 보이면 이미 설치된 거예요! 바로 다음 Step으로!

Step 2. 터미널 처음 열어보기 (3분)

설치 후 "열기" 버튼을 누르세요. (또는 윈도우 키 → "Terminal" 검색)

까만(또는 파란) 창이 뜹니다. 이게 바로 터미널이에요!

📌 무서워하지 마세요! 이건 컴퓨터랑 카톡하는 창이에요.
내가 글을 쓰면 컴퓨터가 대답합니다.

화면에 이런 비슷한 글자가 보일 거예요:

PS C:\Users\여러분이름>

"나는 준비됐어, 명령을 내려줘!"라는 뜻이에요.

Step 3. 첫 번째 명령어 입력해보기 (2분)

아래 명령어를 정확히 따라 입력하고 Enter 키를 누르세요:

echo "안녕, 나는 터미널이야!"

화면에 이렇게 나와야 해요:

안녕, 나는 터미널이야!

🎉 축하해요! 방금 컴퓨터에게 첫 번째 명령을 내렸습니다!

Step 4. 터미널의 종류 알아보기 (5분)

📌 윈도우에는 터미널(까만 창)이 여러 종류가 있어요. 비유로 쉽게 이해해봅시다!
1) 명령 프롬프트(cmd) = "일반 자전거" 🚲

윈도우에 처음부터 있는 가장 오래된 터미널. 할 수 있는 것이 적어요.

화면에 C:\Users\이름> 이렇게 보여요.

2) PowerShell(파워셸) = "전동 자전거" 🛵

cmd를 업그레이드한 버전. 윈도우 10/11에 기본 설치.

화면에 PS C:\Users\이름> 이렇게 보여요 (앞에 PS가 붙음).

방금 우리가 연 Windows Terminal이 기본으로 PowerShell을 보여줘요.

3) Git Bash = "전동 킥보드" 🛴 (우리가 앞으로 쓸 것!)

Git을 설치하면 함께 오는 특별한 터미널. Claude Code가 가장 잘 동작하는 환경!

화면에 이름@컴퓨터 MINGW64 ~ 이렇게 보이고, $ 표시로 명령을 기다려요.

📌 결론: 아래에서 바로 Git Bash를 설치할 거예요!

Step 5. Git 설치하기 (10분)

📌 Git을 왜 설치해야 하나요? Git은 개발자에게 필수인 "코드 타임머신"이에요. 3가지 이유:

1) 실수 복구: 코드를 잘못 바꿨을 때 "아까 그 상태로 돌려줘!"가 가능
→ 비유: 게임의 "세이브 포인트". 중간중간 저장해두면 실패해도 다시 시작!

2) 협업: 여러 사람이 같은 프로젝트에서 동시에 작업 가능
→ 비유: 구글 문서에서 여러 명이 동시에 편집하는 것과 비슷

3) Git Bash: Git을 설치하면 "Git Bash"라는 특별한 터미널이 함께 설치!
→ Claude Code는 이 Git Bash에서 가장 잘 동작합니다

브라우저에서 아래 주소로 이동하세요:

https://git-scm.com/downloads/win

"Click here to download" 또는 "64-bit Git for Windows Setup"을 클릭하세요.

다운로드된 파일(Git-숫자-64-bit.exe)을 더블클릭하세요.

"이 앱이 변경하도록 허용하시겠습니까?" → "예"

설치 화면이 나오면:

1) "Next" 버튼을 계속 누르세요 (약 8~10번)
2) 중간에 여러 옵션이 나오지만, 기본값 그대로 두고 "Next"만!
3) 마지막에 "Install" 버튼 클릭
4) 설치 끝나면 "Finish" 클릭

💡 영어 옵션이 많지만 걱정 마세요! "Next → Next → ... → Install → Finish" 순서만 기억!

Step 6. Git Bash 열고 확인하기 (5분)

윈도우 키 → "Git Bash" 검색 → 클릭

까만 창이 열리면서 이런 글자가 보여요:

사용자이름@컴퓨터이름 MINGW64 ~
$

$ 표시가 "명령을 기다리고 있어요"라는 뜻!

📌 앞으로 이 강의에서 "터미널을 열어주세요"라고 하면 Git Bash를 열어주세요!

Git이 잘 설치됐는지 확인해봅시다:

git --version

이런 결과가 나오면 성공:

git version 2.47.1.windows.2

(숫자는 달라도 OK. 숫자가 나오기만 하면 성공!)

프로젝트 폴더도 미리 만들어둡시다:

mkdir ~/etf-dashboard
📌 mkdir = "make directory" = "폴더를 만들어줘"

Step 7. Node.js 설치하기 (7분)

📌 Node.js(노드제이에스)가 뭔가요?
프로그램을 실행시켜주는 "엔진"이에요.
비유: 자동차(프로그램)가 달리려면 엔진(Node.js)이 필요한 것처럼, Claude Code를 돌리려면 Node.js가 먼저 필요해요.

브라우저에서 아래 주소로 이동하세요:

https://nodejs.org

"LTS"라고 적힌 큰 초록색 버튼을 클릭하세요.

📌 LTS = "Long Term Support" = "오래오래 안정적으로 쓸 수 있는 버전". 항상 이걸 선택!

다운로드된 파일을 더블클릭 → "Next → Next → ... → Install → Finish" 순서로 설치!

⚠️ 중요: 설치 후 Git Bash를 껐다가 다시 열어야 합니다!

Git Bash를 새로 열고 확인:

node --version

v로 시작하는 숫자가 나오면 성공! (예: v22.13.1)

Step 8. Claude Code 설치 + 첫 실행! (8분)

📌 Claude Code가 뭔가요?
터미널 안에서 동작하는 AI 비서예요.
비유: 카카오톡으로 똑똑한 친구에게 "이거 만들어줘"라고 부탁하는 것처럼, 터미널에서 Claude에게 프로그램을 만들어달라고 부탁할 수 있어요.

Git Bash에서 아래 명령어를 입력하세요:

npm install -g @anthropic-ai/claude-code
📌 npm install = "설치해줘", -g = "어디서든 쓸 수 있게"

설치에 1~3분 걸려요. "added XX packages"가 보이면 성공!

프로젝트 폴더로 이동하고 Claude를 실행합시다:

cd ~/etf-dashboard
claude

처음 실행하면 로그인이 필요합니다. 화면에 이런 메시지가 나타나요:

? How would you like to authenticate? (Use arrow keys)
> Anthropic Console (OAuth - recommended)
  API Key

키보드 화살표로 "Anthropic Console"이 선택된 상태에서 Enter를 누르세요.

브라우저가 자동으로 열려요.

브라우저에서:

• 계정이 있다면: 로그인
• 계정이 없다면: "Sign up" 클릭 → 구글 계정으로 간편 가입 가능!
• 로그인 후 "Allow" 또는 "허용" 클릭

터미널에 이런 화면이 보이면 성공:

> Welcome to Claude Code!

>

Step 9. Claude에게 첫 인사하기! (3분)

> 표시 옆에 아래처럼 입력하고 Enter:

안녕! 나는 ETF 대시보드를 만들고 싶어. 도와줄 수 있어?

Claude가 한국어로 친절하게 대답해줄 거예요!

🎉 축하해요! 오늘 한 번에 터미널, Git, Node.js, AI 비서까지 만났습니다!

💡 Claude Code를 끝내려면 Ctrl+C 또는 /exit 입력
