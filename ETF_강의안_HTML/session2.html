Python 설치 + YouTube API 키 발급

⏱️ 35분
🎯 이 회차를 마치면: Python이 설치되고, 유튜브에서 데이터를 가져올 수 있는 열쇠(API 키)를 갖게 되고, 데이터 수집 프로그램까지 만들어져요!

Step 1. Python 설치하기 (10분)

📌 Python(파이썬)이 뭔가요?
컴퓨터 프로그래밍 언어예요. 영어처럼 읽기 쉬워서 초보자에게 인기가 많아요.
우리는 데이터를 수집할 때 Python을 사용할 거예요.

브라우저에서 아래 주소로 이동하세요:

https://www.python.org/downloads/

노란색 "Download Python 3.x.x" 버튼을 클릭하세요.

다운로드된 파일을 실행하면 설치 화면이 나옵니다.

⚠️ 가장 중요한 순간!
맨 아래에 "Add python.exe to PATH"라는 체크박스가 보입니다.
이것을 반.드.시. 체크하세요!
(체크 안 하면 나중에 python 명령어가 안 먹어서 큰 문제가 생겨요)

체크한 후 "Install Now"를 클릭하세요.

설치가 끝나면 "Close"를 누르세요.

Git Bash를 껐다가 다시 열고 확인:

python --version

Python 3.x.x가 나오면 성공!

pip --version
📌 pip = Python의 앱스토어. 필요한 재료를 설치하는 도구. 숫자가 나오면 OK!

Step 2. Google Cloud Console 접속 (3분)

📌 API 키가 뭔가요?
"출입증"이에요. 놀이공원에 입장권이 필요하듯, 유튜브 데이터를 가져오려면 구글이 발급해주는 출입증(API 키)이 필요합니다.
이 출입증은 무료예요! (하루 사용 횟수 제한만 있어요)

브라우저에서 아래 주소로 이동하세요:

https://console.cloud.google.com

구글 계정으로 로그인하세요. (지메일 있으면 OK!)

Step 3. 새 프로젝트 만들기 (3분)

화면 왼쪽 위 "프로젝트 선택" (또는 "Select a project") 클릭

"새 프로젝트" (또는 "New Project") 클릭

프로젝트 이름에 아래처럼 입력:

etf-youtube-dashboard

"만들기" (또는 "Create") 클릭. 잠시 기다리면 프로젝트가 만들어집니다.

Step 4. YouTube Data API 활성화 (3분)

왼쪽 메뉴: "API 및 서비스" → "라이브러리"
(영어: "APIs & Services" → "Library")

검색창에 입력:

YouTube Data API v3

"YouTube Data API v3" 클릭 → 파란색 "사용" (Enable) 버튼 클릭

📌 이 단계는 "유튜브 데이터 택배 서비스를 신청합니다"라는 뜻!

Step 5. API 키 만들기 (3분)

왼쪽 메뉴: "API 및 서비스" → "사용자 인증 정보"
(영어: "APIs & Services" → "Credentials")

"+ 사용자 인증 정보 만들기" → "API 키" 클릭

API 키가 생성됩니다! 이런 모양의 긴 글자:

AIzaSyD-xxxxxxxxxxxxxxxxxxxxxxxxx
⚠️ 매우 중요: 이 키를 복사해서 메모장에 저장하세요!
비밀번호처럼 남에게 보여주면 안 됩니다!

Step 6. API 키를 프로젝트에 저장 (3분)

Git Bash에서:

cd ~/etf-dashboard
echo "YOUTUBE_API_KEY=여러분의_API_키_여기에" > .env

("여러분의_API_키_여기에" 부분에 아까 복사한 키를 붙여넣으세요)

📌 .env 파일은 "비밀 메모장"이에요. 프로그램이 필요할 때 여기서 API 키를 꺼내 씁니다.

잘 저장됐는지 확인:

cat .env

결과:

YOUTUBE_API_KEY=AIzaSyD-xxxxxxxxxxxxxxxxxxxxxxxxx

Step 7. Claude에게 데이터 수집 프로그램 만들어달라고 부탁 (5분)

📌 이번 Step의 핵심: 우리가 직접 코드를 쓰지 않아요!
Claude AI에게 "이런 프로그램 만들어줘"라고 말하면, AI가 대신 만들어줍니다.

Claude Code를 실행하세요:

claude

Claude의 > 프롬프트에 아래 내용을 통째로 복사해서 붙여넣으세요:

.env 파일에 YOUTUBE_API_KEY가 저장되어 있어.
이 키를 사용해서 유튜브에서 ETF 관련 영상을 검색하고
데이터를 수집하는 Python 프로그램을 만들어줘.

요구사항:
1. "ETF 투자", "ETF 추천", "ETF 분석" 키워드로 검색
2. 각 키워드당 최신 영상 10개씩 수집
3. 수집할 정보: 영상 제목, 채널명, 조회수, 좋아요 수,
   게시일, 영상 링크
4. 결과를 etf_youtube_data.json 파일로 저장
5. python-dotenv를 사용해서 .env에서 API 키 읽기
6. 실행하면 수집 진행 상황을 보여주기

Enter를 누르면 Claude가 코드를 작성하기 시작합니다!

💡 Claude가 "파일을 만들어도 될까요?" 같은 질문을 하면 "y" 입력 후 Enter

Claude가 작업을 마치면 /exit으로 나온 후 확인:

/exit
ls

이런 파일들이 보여야 해요:

.env
collect_data.py
requirements.txt

✅ 체크포인트

  • python --version 결과에 Python 3.x.x가 나왔나요?
  • YouTube Data API v3를 활성화(Enable) 했나요?
  • cat .env 결과에 YOUTUBE_API_KEY=... 형태가 보이나요?
  • collect_data.py 파일이 만들어졌나요?
